# 🎯 Google Cloud Certified Associate Cloud Engineer (ACE) — Interactive Study Guide

An interactive, responsive, and high-contrast study mind map and 3D flashcard application designed to prepare for the Google Cloud Certified Associate Cloud Engineer (ACE) exam.

👉 **Live Interactive Study App:** **[https://kid1995.github.io/gcp-study-guide/](https://kid1995.github.io/gcp-study-guide/)**

---

## 🎨 App Features

- **Interactive SVG Mind Map:** Navigation of the complete GCP ACE syllabus. Supports click-and-drag panning, double-finger scroll zooming, and node expand/collapse.
- **Legible Light & Dark Themes:** Seamless dark/light theme switching with crystal-clear contrast.
- **WCAG AA Accessible Contrast Colors:** Automatically recalibrates hard-to-read brand colors (like yellow and cyan) to darker, accessible shades in Light Mode.
- **Embedded Practice Quizzes:** High-yield practice questions wired to their specific nodes. Returns immediate feedback and in-depth correct/incorrect explanations.
- **gcloud CLI Cheat Sheet:** Clean, copying-enabled dark console block instructions for primary `gcloud` terminal operations.
- **3D Interactive Flashcards:** Drills key exam traps and metrics (Retention Policies, HA VPN tunnels, VPC Peering transit rules) using responsive 3D card-flip animations.
- **Learning State Preservation:** Progress is saved automatically to your device's `localStorage` so you can resume where you left off.
